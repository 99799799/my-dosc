# 武阔天下

武阔天下是一款基于 Nuxt + Vue + Element-ui + Less + JavaScript 上开发的游戏会员平台，供用户预览各类游戏，游戏分类，收藏游戏，充值下单，扫码、手机、账号登陆，平台活动，积分兑换等功能
