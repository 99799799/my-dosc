# 微蜂私域

微蜂私域项目是基于Vite + Vue3 + Vue-router + Pinia + UnoCSS + TypeScript上进行开发，实现私域用户后台管理系统，查看用户下单记录，商品管理，群产出，企微应用管理，大淘客佣金管理，公众号进粉详情等功能。
